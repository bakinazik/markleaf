Changelog

  • Added bookmark pinning, pinned items stay grouped at the top and self-heal if dragged out of place.
  • Added a "new item position" setting, choose whether new bookmarks go to the start or end of the folder.
  • Import/export now support scope selection (bookmarks, settings, or both) and Netscape HTML format, Chrome/Firefox compatible.
  • Added a settings search bar with a clear button and persistent input focus.
  • Added context menu customization with drag and drop reordering.
  • Added grid view controls for column count, item width, and title visibility.
  • Added toggles for quick actions, scrollbar, and the scroll to top button.
  • Added a new About panel, with version number, store, website and github links.
  • Refactored the icon system into a centralized 'icons' object.
  • Improved breadcrumbs, you can now drag and drop folders or bookmarks onto them to move them.
  • Redesigned Theme System (Browser/Light/Dark/Black/Custom). You can choose from predefined themes or create your own custom theme.
  • Extension Icon customization (Color/Black/White/Gray/Custom). Set a built-in icon or upload your own.
  • Favicon Provider options (Browser/Google/DuckDuckGo). Browser mode loads favicons without external requests while Google and DuckDuckGo can fetch them via their services.
  • Folder-only search setting added. When enabled, search will be limited to the current folder and its subfolders. By default, search applies to all bookmarks.
  • Main folder selection added (Bookmarks Bar / Other Bookmarks).
  • Context menu can now be opened by clicking on empty space.
  • Breadcrumb navigation added (toggle in settings).
  • New context action: add active tabs to a selected location or folder.
  • Search bar now includes a clear "x" button when input is present.
  • Bug fix: missing favicons (unvisited sites) now load instantly without requiring extension reopen.
  • Multi-select mode with bulk actions.
  • Trash system with restore and auto-delete.
  • Import / Export (full backup & restore).
  • Improved search performance and responsiveness.
  • Theme control and UI customization options.
  • Other details available on GitHub
  • Markleaf is now open source! GitHub
  • You can navigate search results using the arrow keys. Pressing the up and down arrows moves through the list; pressing right or Enter opens the selected item; pressing left or Tab goes up a folder. Moving the mouse cancels the keyboard selection.
  • Content loads at least 30% faster. Folder item counts and favicon previews are now calculated from already-loaded data instead of making separate requests for each item.
  • A three-column grid layout has been implemented.
  • Right-clicking on an empty area no longer opens the browser's native menu the custom menu closes instead.
  • The context menu now plays a closing animation, just like it does when opening.
  • Moving a folder into itself or one of its own subfolders is no longer possible.
  • Scrolling back to the top is now smoother. It now uses the browser's built-in scroll engine instead of a custom animation, resulting in a more natural and consistent feel.
  • Context menu animations have been improved.
  • Browsers that do not support the browser's default favicon feature (such as Opera) automatically use Google's favicon service. This parameter will be removed in the future if they adopt the newer system.
  • The list view has been redesigned with a modern design approach.
  • You can now open the context menu by right-clicking a list item.
  • Folders now display the total number of bookmarks they contain.
  • Shortcut improvements.
  • Added a “Back to Top” button.
  • Scrollbar is now visible.
  • Performance and design improvements.
